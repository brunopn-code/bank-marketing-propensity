# Data

This project uses the **UCI Bank Marketing dataset**, which contains data from direct marketing campaigns of a Portuguese banking institution.

The classification goal is to predict whether a client subscribed to a term deposit.

## Source

- Dataset: Bank Marketing
- Source: UCI Machine Learning Repository
